CE-316 Project User Guide

Introduction

Welcome to the CE-316 Project, a tool designed to streamline the process of compiling, executing, and comparing student submissions. This guide will help you understand how to use the application effectively.

Main Interface

The application consists of several components that help you manage student submissions and configurations.

Menu Bar

Project Menu
Contains options for creating a new project and quitting the application.
Configuration Menu
Allows you to create new configurations or open existing ones.
Help Menu
Provides access to this user guide and information about the application.

Project Workflow

Creating a New Project

To create a new project:

  1. Click on Project > New Project or use shortcut Ctrl+N
  2. Specify the project details including path to student submissions
  3. Select or create a configuration for compilation and execution
  4. Click "Create" to initialize the project

Managing Configurations

Configurations determine how submissions are compiled and executed.

  • Create a new configuration via Configuration > New Configuration
  • Open and edit existing configurations via Configuration > Open Configuration
Tip: Create different configurations for different programming languages or assignment requirements.

Features and Function Buttons

Compilation

The system supports both compiled and interpreted languages:

Compile Submissions
Compiles all student submissions in parallel according to the configuration.

Execution

Execute Submissions
Runs the compiled submissions and captures output and errors.

Comparison

Compare Results
Compares student outputs against a reference solution to determine if they pass or fail.

Status Indicators

Throughout the process, student submissions will have different status indicators:

Status Description
READY Successfully compiled and ready for execution
COMPILING Currently being compiled
EXECUTING Currently being executed
COMPLETED Execution completed successfully
PASSED Output matches reference solution
FAILED Output does not match reference solution
ERROR An error occurred during compilation or execution

Troubleshooting

If you encounter issues:

  • Check the error message in the compilation or execution results
  • Verify that the configuration parameters match the requirements of the source code
  • Ensure that compiler paths and execution commands are correctly specified
  • For interpreted languages, make sure the interpreter path is correctly configured
